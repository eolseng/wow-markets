# Code of conduct

We pledge to make participation in this project a harassment-free experience
for everyone. Be respectful, constructive, and considerate. Unacceptable
behavior includes harassment, discrimination, threats, sexualized attention,
publishing private information, or sustained disruption.

Report conduct concerns privately using the contact route in
[SECURITY.md](SECURITY.md). Maintainers may remove contributions or restrict
participation when behavior harms the community. Enforcement decisions will
be made fairly and with respect for the privacy of everyone involved.

This policy is based on the Contributor Covenant, version 2.1.
